# ワンちゃんの縁 (PickPet)

### Setup
#### 1. Clone the Repository
```bash
git clone https://github.com/yuanyuanweng/DBMS_G7.git
cd DMBS_G7
```

#### 2. Venv/Conda (To be updated)  

#### 3. Install Requirements
```bash
pip install -r requirements.txt
```

#### 4. Start Server  
```bash
python run.py
```

#### 5. Open in Browser
Go to: http://127.0.0.1:5000/
